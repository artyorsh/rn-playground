## Setup

```bash
yarn
```

```bash
npx pod-install ios
# or pod install --project-directory=ios
```

```bash
cp .env.example .env
```

## Running

Start the Metro bundler

```bash
yarn start
```

Run the app

```bash
yarn ios
# or yarn android
```
